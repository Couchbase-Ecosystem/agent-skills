Function names are used to apply a function to values, to values at a specified path, or to values derived from a DISTINCT clause.
Function names are case insensitive.

Here are the categories of {sqlpp} functions:

* [Aggregate Functions](n1ql-language-reference/aggregatefun.adoc)
* [Array Functions](n1ql-language-reference/arrayfun.adoc)
* [Bitwise Functions](n1ql-language-reference/bitwisefun.adoc)
* [Comparison Functions](n1ql-language-reference/comparisonfun.adoc)
* [Conditional Functions for Unknowns](n1ql-language-reference/condfununknown.adoc)
* [Conditional Functions for Numbers](n1ql-language-reference/condfunnum.adoc)
* [Date Functions](n1ql-language-reference/datefun.adoc)
* [JSON Functions](n1ql-language-reference/jsonfun.adoc)
* [Miscellaneous Utility Functions](n1ql-language-reference/metafun.adoc)
* [Number Functions](n1ql-language-reference/numericfun.adoc)

* [Pattern-Matching Functions](n1ql-language-reference/patternmatchingfun.adoc)
* [Search Functions](n1ql-language-reference/searchfun.adoc)
* [String Functions](n1ql-language-reference/stringfun.adoc)
* [Token Functions](n1ql-language-reference/tokenfun.adoc)
* [Type Functions](n1ql-language-reference/typefun.adoc)
